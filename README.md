This project is an interactive AI chatbot web application built using Streamlit, powered by Google Gemini 2.5 Flash via LangChain. It allows users to ask questions and receive intelligent, real-time responses through a clean and simple UI.

The app also integrates LangSmith tracing for monitoring and debugging LLM interactions.

🚀 Features

- 💬 Real-time conversational AI chatbot

- ⚡ Fast responses using Gemini 2.5 Flash

- 🔗 Built with LangChain Expression Language (LCEL)

- 🎯 Prompt templating for structured responses

- 📊 LangSmith tracing for observability

- 🖥️ Simple and interactive Streamlit UI

- ⚙️ Error handling for robust performance

🛠️ Tech Stack

- Frontend/UI: Streamlit

- LLM: Google Gemini 2.5 Flash

- Framework: LangChain

- Tracing & Monitoring: LangSmith

- Language: Python

💡 Usage

- Enter your question in the input field

- The chatbot processes it using Gemini 2.5 Flash

- View the generated response instantly
